¿Qué es un generador de contraseñas?

Es una herramienta que genera una contraseña de forma automática, en base a los parámetros que usted define.

Para usar SimplyKey, usted solo necesita recordar:

  • Un nivel de seguridad, que corresponde a un dígito. Entre más grande su tamaño, más segura será la contraseña.
  • Un texto (puede usar espacios) sencillo de recordar.
  • Está permitido usar como texto letras (mayúsculas y minúsculas), dígitos y carácteres. Si ingresa algun carácter no válido, ¡SimplyKey se lo indicará!.

Así, podrá generar su nueva contraseña.

¿Obtuvo su contraseña y no recuerda que texto utilizó para generarla?

SimplyKey le permite restaurar su contraseña al texto usado.

Utilice la opción Restaurar texto, para obtener el texto usado para generar la contraseña. Sólo debe recordar su nivel de seguridad.
